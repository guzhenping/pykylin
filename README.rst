Kylin DBAPI Driver and Sqlalchemy Dialect
==============================

前言
------------

主要是为了研究kylin+redash的OLAP解决方案。比较有针对性。

Installation
------------

To install this driver run::

    $ pip install pykylin

or from source::

    $ pip install -r ./requirements.txt
    $ python setup.py install
